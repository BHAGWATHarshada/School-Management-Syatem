# School-Management-Syatem
his project is a multi-role authentication system that redirects users to role-specific dashboards based on their login credentials. It provides distinct interfaces and functionalities for Teachers, Parents, and Students, ensuring a seamless experience for each user type.

Features
Teacher Dashboard:

Class management tools
Timetable scheduling
Announcements section
Contact features for communication
Parent & Student Dashboards:

Student details overview
Fee payment system
Performance tracking
Query submission for easy communication
